這個裝置不支援 File System API，無法讀取資料夾。
請改用桌機版 Chrome 或 Edge 開啟本頁。

把你的編織圖資料夾變成可瀏覽的畫廊！

第一次使用，請按下面的按鈕，選取要預覽的資料夾，基於瀏覽器的安全規定，網頁不能自己讀取你電腦裡的檔案，因此要由你親自授權一次。

所有內容以你的資料夾為準。網頁只是把資料夾裡的內容呈現給你看，不會搬動或修改、刪除你既有的檔案。

※ 請使用 Chrome 或 Edge 開啟。所有檔案只在你電腦本機讀取，不會上傳到任何地方。

換的只是網頁顯示哪個資料夾的內容，你電腦裡的檔案完全不會被動到。網頁之後會從新資料夾重新讀取要顯示什麼。

🛟 舊資料夾裡的東西都還在，沒有跟著消失：

  • PDF／圖片本來就是你的
  • 之前在網頁裡收藏的網址，都還在舊資料夾根目錄的 links.md
  • 網址縮圖在舊資料夾的 thumbs/ 子資料夾
  • 這兩個是網頁幫你建的，但它們屬於你
  • 想把收藏帶到新資料夾？在檔案總管把 links.md 和 thumbs/ 搬過去即可

🔄 如果新資料夾本來就有 links.md 和 thumbs/（比如之前也用網頁整理過、或剛剛從別的地方搬過去），網頁會直接讀進來，不會覆蓋。
